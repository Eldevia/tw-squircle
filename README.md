# [!WIP!] Tailwind CSS Squircle Utilities

A squircle plugin for Tailwind CSS. This package provides a set of predefined `squircle-*` classes that you can use in your project.

## Installation

You can install the package using npm, pnpm, or bun:

```bash
npm install tw-squircle
```

```bash
pnpm add tw-squircle
```

```bash
bun add tw-squircle
```

## Usage

To use the plugin, import the `tw-squircle` file into your main `tailwind.css` file.

```css
@import "tailwindcss";
@import "tw-squircle";

/* Your other styles here */
```

This will make the following utility classes available:

- `squircle-16`
- `squircle-stroke-16`
- `squircle-16-12`
- `squircle-24`
- `squircle-24-16`
- `squircle-28`
- `squircle-36`
- `squircle-36-24`
- `squircle-48`
- `squircle-48-32`
- `squircle-32-48`
- `squircle-60`
